# HR Analytics Dashboard 📊

## 📌 Project Overview
This project focuses on analyzing employee data to identify attrition trends and workforce insights.

## 🔍 Key Insights
- Total Employees: 1470
- Attrition Rate: 16.12%
- Active Employees: 1233
- Average Age: 37

## 📊 Features
- Department-wise Attrition
- Age Group Analysis
- Gender-based Insights
- Job Satisfaction Ratings
- Education Field Analysis

## 🛠️ Tools Used
- Excel
- Power BI
